Ideas of this template

 The basic ideas of this template are:
  • As much as possible customizable without php or css skills
  • Use your own colors and designs
  • Place your own logo (general or page dependend)
  • Put up a photo slider using module
  • Hugh amount of positions to use for modules like you want

Areas & positions
position-13 (logo) position-0 (search)
position-1 (menu)
position-2 (breadcrumb)
position-7 (module left top) position-12 (content header) position-6 (module right top)
position-4 (module left center) content position-8 (module right center)
position-5 (module left bottom) position-14 (content footer) position-3 (module right bottom)
position-9 (footer left) position-10 (footer center) position-11 (footer right)

Placing a logo
  1. In the Administration user interface go to: Extensions - Module Manager
  2. Press the button "New"
  3. Select the entry "Custom HTML"
  4. Enter a Title (like Logo)
  5. Design your logo like an article and place it on position-13
  6. Save and you are finished
Multiple logos for different pages:
  1. Create all your logos like described above
  2. Use "Menu Assignment" to specify on which page this logo should be shown

Placing a copyright

 Use the same steps like creating a logo but use position-10 instead.

Placing a photo slider on the frontpage
  1. Select your favorite slider module from here
  2. Place it on position-12 (Content header)
  3. Select on "Menu Assignment": Only on the pages selected
  4. Unselect all and select your start page
