Ein Logo platzieren
  1. Im Administrations-Bereich navigieren nach: Erweiterungen - Module
  2. Die Schaltfläche "Neu" anklicken
  3. Den Eintrag "Benutzerdefiniertes HTML" wählen
  4. Titel eingeben (z.B. Logo)
  5. Designen Sie Ihr Logo wie einen Artikel und platzieren Sie es auf position-13 (Logo)
  6. Speichern und fertig
Mehrere Logos für verschiedene Seiten:
  1. Erstellen sie Ihre Logos wie oben beschrieben
  2. Benutzen Sie "Menüzuweisung" um die Logos den Seiten zuzuweisen

Impressum und Urheberrechte platzieren

 Benutzen Sie dieselben Schritte wie ein Logo erstellen aber platzieren Sie das Modul auf position-10 (Fussbereich Mitte).

Eine Fotopräsentation auf der Haupseite platzieren
  1. Installieren Sie eines der Fotopräsentation-Module von hier
  2. Platzieren Sie das Modul auf position-12 (Inhalt Kopfbereich)
  3. Selectieren Sie "Nur auf den gewählten Seiten" unter "Menüzuweisung"
  4. Alle Seiten deselektieren und die Startseite selektieren
